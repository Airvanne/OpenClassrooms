OPEN CLASSROOMS

Bonjour, ce repo est destiné à un exercice sur OpenClassrooms !

Il y a deux fichiers .rb qui sont en fait des jeux sur terminal. Il nécessite d'avoir ruby d'installer sur son ordinateur.

Si c'est le cas vous pouvez donc :

- avec jeu.rb, participez aux aventures de Jean-Michel Paladin !
	(il vous suffit de taper `ruby jeu.rb` dans votre terminal puis suivre les consignes)

- avec pyramide.rb, construisez la pyramide de vos rêves ! (lancez `ruby pyramide.rb` dans votre terminal et suivez la consigne)
